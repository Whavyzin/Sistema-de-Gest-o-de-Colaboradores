# RHyos - Sistema de Gestão de Colaboradores

Sobre o Projeto:
O RHyos é um sistema web desenvolvido em PHP e MySQL para gerenciamento de colaboradores.

O sistema permite:
- Adicionar funcionários
- Editar funcionários
- Excluir funcionários
- Visualizar funcionários
- Login e Logout de usuários

---

Tecnologias Utilizadas:
- PHP
- MySQL
- HTML
- CSS
- InfinityFree
- GitHub
